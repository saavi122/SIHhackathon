# Rural Health Connect

Rural Health Connect is a healthcare portal prototype designed for rural communities.  
It provides easy access to medical services such as consultations, emergency help, and medicine vendors through a single platform.  
The entire project is built as a single responsive HTML file using HTML, CSS, and JavaScript.  

---

## Features

- Login and Signup system with role selection (Patient, Doctor, Pharmacy)  
- Emergency help section available on the login page (Nearest Hospital, Ambulance, Urgent Checkup)  
- Dashboard with core healthcare options:
  - Book Consultant  
  - Video Consultation  
  - Medicine Vendor  
- Floating AI chat icon for instant help  
- Responsive design for mobile and desktop  
- Single-page navigation without reload
